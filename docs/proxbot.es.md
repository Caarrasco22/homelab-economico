# ProxBot v1

ProxBot v1 is a Discord bot under development for learning homelab automation and management.

## Goals

- Learn how to interact with homelab services from Discord.
- Centralize basic status checks.
- Move hardcoded values to a configuration file.
- Keep the bot simple, safe and maintainable.

## Current status

- Dedicated Debian LXC container prepared.
- Basic development environment in progress.
- Configuration-based approach planned using `config.json`.

## Planned features

- Check service status.
- Show basic homelab information.
- Run safe predefined commands.
- Avoid exposing sensitive data in Discord responses.
