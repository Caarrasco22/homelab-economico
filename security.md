# Security Policy

This repository documents a personal homelab environment for learning and portfolio purposes.

No real credentials, private keys, tokens, exact remote-access addresses, public endpoints, usernames, MAC addresses, serial numbers or sensitive configuration files should be published here.

If sensitive information is accidentally exposed, it should be removed from the repository history as soon as possible.

---

# Política de seguridad

Este repositorio documenta un entorno de homelab personal con fines de aprendizaje y portfolio.

No deben publicarse credenciales reales, claves privadas, tokens, direcciones exactas de acceso remoto, endpoints públicos, nombres de usuario, direcciones MAC, números de serie ni archivos de configuración sensibles.

Si se expone información sensible por accidente, debe eliminarse del historial del repositorio lo antes posible.
